# Playwright Automation Project

This project is an end-to-end test automation framework built using Playwright and JavaScript.

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Features
- UI automation testing
- Page Object Model (POM)
- Cross-browser testing
- Reusable test scripts

## How to Run
1. Install dependencies  
   npm install

2. Run tests  
   npx playwright test

3. View report  
   npx playwright show-report
